\# Gwent - Jeu de cartes



Reproduction du jeu de cartes Gwent (The Witcher 3) en C# Windows Forms.



\## Fonctionnalités



\- Partie locale (2 joueurs sur le même PC)

\- Partie en réseau (héberger / rejoindre)

\- Sauvegarde et chargement de parties

\- 4 factions jouables avec pouvoirs passifs uniques

\- Toutes les mécaniques de cartes (Espion, Médic, Météo, etc.) # pas à 100% #



\## Comment jouer



1\. Téléchargez la dernière release

2\. Extrayez le ZIP

3\. Lancez `Gwent.exe`



\## Mode réseau



\- \*\*Héberger\*\* : Cliquez sur "Héberger" et partagez votre IP

\- \*\*Rejoindre\*\* : Entrez l'IP de l'hôte et le port (12345)
